# Showcase

The showcase PDF is the broad reference document for Paper's public authoring paths. It combines HTML translation, typed rows and columns, reusable components, tables, codes, headers, footers, page numbers, and metadata in one generated file.

## PDF Generated
```pdf
	assets/pdf/showcase.pdf
```

## Rebuild
```bash
cd examples
go run ./cmd/paper-showcase ../docs/assets/pdf/showcase.pdf
```
